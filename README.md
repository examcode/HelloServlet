HelloServlet
============
Servlet, JSP Example

### Environment
* centos 5.8
* tomcat 7.0

### Checkout
* git clone https://github.com/examcode/HelloServlet.git

### Build
* mvn clean package

### Start/Stop
	..tomcat/bin/shutdown.sh
	..tomcat/bin/startup.sh

### Test
	http://...:8080/test.do?name=최제승
